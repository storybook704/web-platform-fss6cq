Change ☑

Feeling

ကမ်းစပ် နဲ့ ရေလှိုင်း
ဘာ့ကြောင့် နီးစပ်ပြီးမှ
ပြန်ဝေးသွားလဲ နင်ပဲပြန်ဖြေ။

အဲ့သလိုပဲ သဘာ၀တရားဆိုတာလည်း
ပေးသင့်သလောက်ပေးခဲ့ရတာထက်
ရသင့်သလောက်ပြန်မရခဲ့ကြတာ …

ရှင်းပါတယ် – ကမ္ဘာမြေရဲ့ နိယာမတွေတောင် မတူညီကွဲပြားခြားနားကြ သေးတာပဲ – ငါဆိုတာ အတောင်ညောင်းတဲ့ ငှက်ငယ်မို့ ခေတ္တနားခိုရာ မင်းသစ်ပင်အကိုင်းအခက် က မကြိုဆိုခဲ့တာကို မာယာများတဲ့ သစ်ပင်ပါလို့ တံဆိပ်ခတ်နှိပ် ချိတ်ဆွဲပါရစေ …

မြင့်မိုရ်တောင် ကို ဇီးစေ့နဲ့ပေါက်သလို အနောက်အရပ်က နေထွက်ပြီး အရှေ့အရပ်မှာ နေဝင်ပါလို့ မြောက်အရပ်က ဓူဝံကြယ်ကို တောင်အရပ်သို့ ရွေ့ပစ်နိုင်တဲ့ ဦးကိုင်ဇာသီချင်းထဲကလို System Set up တွေကို တန်ခိုးရှင်တွေက သနားသောအားဖြင့် changed လုပ်ချင် /လုပ်နိုင်လိမ့်မယ် …

ငါအသိဉာဏ်နဲ့ နှလုံးသားက Alarm လုပ် တာ ဒီကမ္ဘာတင်မက – သံသရာတစ်ဆုံး အထိ ပက်ပက်စက်စက် နင့်ကို ငါမုန်းတယ်…………!

(ရာဇဝင်တွေသိပ်ရိုင်းဖူးခဲ့တဲ့)
                  #သက်ထက်ဇော်_တီဇစ်
                 
              ၁၈ရက် / မတ်လ /၂၀၂၂                                   
                     သောကြာနေ့
                – မနတ် ၄ နာရီ / ၂၁ မီးနစ်

– ဘူတာ –

Feeling

ကိုယ်မပိုင်တဲ့ ဘဝထဲရောက်လာခဲ့ကြတယ်အတည်လို့ပြောပြီး ဧည်သည်တွေလို
ပြန်ထွက်သွားကြ ။ဪ ... ငါဟာ ဂိတ်ဆုံးမဟုတ်တဲ့ ခေတ္တရပ်နားခွင့်သာရတဲ့ ဘူတာ။

ဆုံခွင့်သာပါပြီး / ပေါင်းစပ်ခွင့်မရှိတဲ့ ဘဝတွေ မျဉ်ပြိုင်လို ဆုံမှတ်မရှိကြတာဆို အစေားကတည်း
က ငါဟာ တာမထွက်ခဲ့ပါဘူး။ဪ ... ငါဟာ ဂိတ်ဆုံးမဟုတ်တဲ့ ခေတ္တရပ်နားခွင့်သာရတဲ့ ဘူတာ။

ဒီနှစ် ရာသီဥတုတွေသိပ် ဖောက်ပြန်တာပဲ လူတွေရဲ့ စိတ်တွေလည်း ပုသင်ညိုထက် ပိုမြန်တဲ့ အရောင်ပြောင်းတာတွေ ငါကြုံနေရတယ်။ဪ ... ငါဟာ ဂိတ်ဆုံးမဟုတ်တဲ့ ခေတ္တရပ်နားခွင့်သာရတဲ့ ဘူတာ။

ကမ္ဘာကျော်တွေလည်းသေတယ် / ပျော်ရွှင်စရာတွေရယ် ပျောက်ရှခဲ့ရတယ် – လူတိုင်းရဲ့မျက်နှာမှာလည်း Mask တွေနဲ့ အားလုံးဟာ ဟန်ဆောင်မှုတွေကို ဖုံးကွယ်ကာ ပြုံးရယ်ကြတယ်။
နင်လည်း ထို့အတူပါပဲ ဪ ... ငါဟာ ဂိတ်ဆုံးမဟုတ်တဲ့ ခေတ္တရပ်နားခွင့်သာရတဲ့ ဘူတာ။

(ရာဇဝင်တွေသိပ်ရိုင်းဖူးခဲ့တဲ့)
#သက်ထက်ဇော်_တီဇစ်

၁၇ ရက် / မတ်လ /၂၀၂၂
ကြာသာပတေးနေ့
– ည၂ နာရီ / ၂၁ မီးနစ်

(မူလသစ်ကိုင်းက ကြွေလွှင့်သစ်ရွက်)

Feeling

ရှေးစာဆို ကဝိ ပညာရှိတွေရေးကြတယ် ရည်းစားဟောင်းဆိုတာ (ထွက်ခွာသွားတဲ့ ရထားတစ်စင်း လို ) (ထွက်ခွာသွားတဲ့ သင်္ဘောကြီးတွေ လို) တဲ့

ကျနော့်အတွက်တော့  ရည်းစားဟောင်းဆိုတာ ကြွေသွားတဲ့ သစ်ရွက်ခြောက်တွေလိုပါပဲ

(ထွက်ခွါသွားတဲ့ ရထားတွေ/သင်္ဘောကြီးတွေဆိုတာ အချိန်တန်ရင်ဖြစ်ဖြစ် / လမ်းဆုံးရင်ဖြစ်ဖြစ် နဂိုဘူတာ /နဂို မီးပြတိုက်ဆီ လမ်းကြုံရင် ပြန်ရောက်ဆုံနိုင်ပါသေးတယ်)

(မြေခကြွေလွင့်သွားရှာတဲ့
သစ်ရွက်တွေဆိုတာကတော့
အကိုင်းအခက်ကနေ မြေပေါ်သက်ဆင်းသွားပြီးတဲ့အခါမှာတော့ –  မူလသစ်ကိုင်းမှာ ပြန်နေဖိုဆိုတာ မဖြစ်နိုင်တော့တဲ့အတွေးတွေပဲလေ)


(ရာဇဝင်တွေသိပ်ရိုင်းဖူးခဲ့တဲ့)
#သက်ထက်ဇော်_တီဇစ်

၁၀ရက် / ဖေဖော်ဝါရီ /၂၀၂၂
ကြာသာပတေးနေ့
– ည၁၀ နာရီ / ၃၆ မီးနစ်
